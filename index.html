This project is under the Open Innovation theme:

Streaming and curating content for DishTV online using Facial Recognition and Gender Analysis
Brief Intro

At a glance, this project is just a simple web application with one goal in mind - craving for something new. End-users can create their accounts on the portal to find exclusive curated content based on their Gender and Age.
And, apart from providing curated content based on gender (and age), users can stream the videos in real-time. Plus, we are putting every user's credential hash on a central blockchain for security purposes.

Using a JS-library called nude.js (no pun intended) every video being uploaded to the portal can be moderated to be filtered for unsafe, offensive, vulgar and adult content.
So, how does it work?

  • End-user comes on the portal and initiates account creation.
  • A channel package is selected and compliance for using our curated content is duly noted.
  • User credentials are hashed together on a central blockchain for security (and sales growth) purposes.
  • Appropriate services are served depending on the previous choices made.
The Tech Stack

Developed in MEAN Stack and hosted on Heroku, it's by far the modern development environment. The web-app uses browser-cache as its own standalone service worker aka offline content provider. The facial recognition system uses the GCP Cloud Vision API to provide JSON-formatted data about user sex and age.
